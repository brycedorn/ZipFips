# ZipFips

* [Homepage](https://rubygems.org/gems/zipfips)
* [Documentation](http://rubydoc.info/gems/zipfips/frames)

## Description

Easily convert between ZIP codes and FIPS codes

## Usage

    require 'zipfips'

    12345.to_fips #=> 36093
    36093.to_zip  #=> 12345

    36093.zipfips #=> 12345

## Install

    $ gem install zipfips
